# Frequently Asked Questions

## Where are the restroom and smoking room?

They are available on each floor. If you are not sure, please ask staff.

## How do I turn on the PC?

After taking your seat, press the power button on the PC tower.

## How can I check my remaining time and balance?

You can check them in the billing system at the bottom-right of the PC screen.

## Can I borrow a mouse, controller, or other equipment?

Yes. Some equipment can be borrowed for free. Please ask staff.

## Can I temporarily leave the store?

Yes. The PC can stay locked, but usage time will still be charged while you are away.

## Can I reserve a seat in advance?

Reservations are usually not supported. Please visit the store directly.

## Do I need an ID document to register as a member?

Yes. Please prepare a valid ID such as a passport, student ID, or driver's license.

## When are seats usually available if the store is full?

Seats are usually more likely to become available after 22:00.

## Can I adjust the private room lighting myself?

Yes. There is a corresponding switch inside the private room.

## What games can I play in the store?

Please check the "Game List" document on the desktop. It lists the currently supported games and platforms.

## What should I do if I have a problem?

Please contact staff directly for assistance.

# Equipment Issues

## What should I do if the mouse or keyboard is not working?

1. Connect the keyboard or mouse to power and confirm it has battery or charge.
2. Check whether the keyboard or mouse receiver is firmly plugged in.
3. Check the device switches:
   - For the keyboard, confirm it is set to "2.4G" mode and "Win" mode
   - For the mouse, confirm the power switch is on
4. If none of the above works, please contact the network/admin staff.

## What should I do if there is a headset problem?

If the headset microphone has no sound: check whether the mute button on the rear-left side of the headset is enabled.

If the headset has no sound:

1. Move the volume wheel on the rear-right side of the headset up and down.
2. Unplug and reconnect the headset cable.
3. If none of the above works, please contact the network/admin staff.

## What should I do if the monitor has a black screen or blue screen?

1. Check whether the power switch on the lower-right back side of the monitor is on.
2. Check whether the monitor input mode is correct. The correct mode is DP*; do not select HDMIA.
3. Check whether the monitor cable is loose.
4. If none of the above works, please contact the network/admin staff.

# Game FAQ

## What should I do if I cannot log in to China server games?

Please confirm:

- You selected the Chinese system when starting the PC
- You are using the accelerator from the desktop "Game Menu"
- Recommended accelerators include UU International, Leigod, and VK
- Xunyou accelerator is not recommended

## What should I do if I cannot enter China server VALORANT?

Please confirm:

- The current system is Chinese
- You started it from the "Game Menu" or "WeGame Internet Cafe Edition"
- Do not turn on an accelerator
- If TPM / HVCI appears, it is usually an account location-risk issue

## What should I do if I cannot enter Japan server VALORANT?

Please confirm:

- The current system is Japanese
- You logged in from Riot Client on the desktop
- If TPM / HVCI appears, it is usually an account issue

## What should I do if I cannot log in to Delta Force?

Please confirm:

- The Steam version can use the Japanese system
- The China server version must use the Chinese system
- Start the game from WeGame
- During shader preloading, do not move the mouse, switch screens, or start an accelerator

## What should I do if League of Legends has frame drops or teleport-like movement?

In-game settings:

- Video: turn off "Wait for Vertical Sync"
- Game: turn off "Use Movement Prediction"

## What should I do if a game download or update fails?

Try clicking the update button repeatedly, or restart the game platform.

## What should I do if I cannot find a game in the game menu?

Open the "Game Menu" on the desktop and search for it.

## Can I install my own game?

In some cases, yes. Please ask staff first to confirm disk space and permissions.

## Why do some games require switching the system language?

Some games' anti-cheat systems check the system language:

- China server games usually require the Chinese system
- Japan server games usually require the Japanese system

## What should I do if there is an account abnormality such as TPM / HVCI?

In most cases, this is an account location-risk control issue, not a PC fault. We recommend changing accounts, trying again later, or contacting the game platform's support.
